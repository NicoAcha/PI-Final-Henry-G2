# Automatización de actualización de ventas

El proyecto incluye un proceso automatizado que permite la incorporación de nuevas ventas.

Este proceso:
- Carga nuevos archivos de ventas.
- Los integra con el histórico.
- Actualiza las tablas analíticas.
- Permite refrescar el dashboard en Power BI.

Los scripts se encuentran en la carpeta `scripts_ETL`.
